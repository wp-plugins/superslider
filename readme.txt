== Description ==

SuperSlider is in production, you can get the following plugins in advance from this suit: SuperSlider-Show, SuperSlider-Menu, SuperSlider-Milk(coming soon). Plugins that use [Mootools](http://mootools.net/ "Your favorite javascript framework") 1.2 javascript. Highly configurable, theme based design, css based animations. Shortcode system on post and page screens and or Widgets. Degrades gracefully with javascript turned off, or plugin removed / disabled.
