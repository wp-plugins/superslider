== Description ==

SuperSlider contains: SuperSlider-Show, SuperSlider-Menu, SuperSlider-Milk(coming soon). Plugins that use [Mootools](http://mootools.net/ "Your favorite javascript framework") 1.2 javascript. Highly configurable, theme based design, css based animations. Shortcode system on post and page screens and or Widgets. Degrades gracefully with javascript turned off, or plugin removed / disabled.

== Installation ==

* Unpack contents to wp-content/plugins/ into a **superslider-show** directory
* Activate the plugin,
* Configure global settings for plugins under > settings > ss-Show / ss-Menu
* (optional) move SuperSlider-Show / Menu  plugin sub folder plugin-data to your wp-content folder,
	under  > settings > ss-Show > option group, File Storage - Loading Options
	select "Load css from plugin-data folder, see side note. (Recommended)". This will
	prevent plugin uploads from over writing any css changes you may have made.
